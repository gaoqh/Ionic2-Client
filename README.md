# Ionic2-demo
### build environment with npm and add plugins.
#####$ cd Ionic2-demo
#####$ npm install
#####$ ionic serve 

### add platforms
#####$ ionic platform add android
#####$ ionic platform add ios
### build
#####$ ionic build ios
#####$ ionic build android
